# CardGame

## 게임 제목 : 기억력 카드 매치 🃏

### 게임설명 🎮
- 이 게음은 뒤집혀 있는 카드를 하나씩 뒤집으며 같은 그림을 가진 카드를 짝짓는 기억력 게임입니다. 카드의 위치를 기억하고 빠르게 짝을 맞추어 승리하세요!

### 특징 ⭐
- 🔍 간단하고 직관적인 인터페이스
- 🧩 카드 짝 맞추기 도전으로 기억력 향상
- ⏳ 시간 제한 없이 여유롭게 플레이 가능
- 👶👵 모든 연령대가 즐길 수 있는 쉬운 규칙

### 게임목표 🎯
🚀 모든 카드를 짝맞추어 최소한의 시도로 게임을 완료하세요! 🚀
